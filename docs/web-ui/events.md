# Web UI — Events

The Events page shows upcoming calendar events for your server in a read-only table.

---

## Accessing events

1. Log in to the [web dashboard](overview.md).
2. Select your server from the dashboard.
3. Click **Events** in the sidebar.

---

## What's shown

The table displays upcoming events with:

| Column | Description |
|---|---|
| **Title** | Name of the event. |
| **Description** | Optional longer description. |
| **Start** | Start date and time (shown in your configured server timezone). |
| **End** | End date and time. |

Only **upcoming** events are shown — past events are not listed.

---

## Creating events

Events are currently created programmatically (e.g. via the scheduler or a connected integration). Managing events directly from the web UI is not yet supported.
