## Roadmap

Planned features and improvements in rough priority order:

### Near-term
- **Document upload via web UI** — upload files directly from the dashboard rather than via Discord.
- **Improved event management** — create and edit calendar events from the web dashboard.
- **Glossary bulk import** — upload a CSV or YAML file to seed the glossary in one step.
- **Per-channel context settings** — configure context window size differently per channel.

### Medium-term
- **Multi-model support** — switch between Anthropic models per server or per channel.
- **Encounter generator** — built-in tool that generates balanced random encounters based on party level and terrain.
- **NPC generator** — quick NPC name, personality, and backstory generation on demand.
- **Session recap command** — generate a formatted session summary from the channel history.

### Long-term
- **Foundry VTT integration** — sync actors and items directly from a running Foundry instance.
- **Voice channel support** — text-to-speech bot presence for ambient narration.
- **Campaign wiki export** — export the glossary and indexed documents to a static HTML wiki.
